## Setup

1. Install [Node.js](https://nodejs.org/en/)

2. Open with [VS Code](https://code.visualstudio.com/)

3. Install the suggested extensions when prompted (Recommended)

4. Install the required packages

   ```sh
   npm i
   ```

   > Select `Current TS File` launch option. **F5** runs the file you're
   > focusing in debug mode.

## Debug

Use the `Current File` launch config to run the focused typescript file.

### Testing

There are tests on every task. You can run them all with

```sh
npm test
```

With coverage report

```sh
npm run test:coverage
```

or if you installed the recommended mocha extension you can run them with that.

### Benchmarking

Run the `bench` files as normal scripts. The resulting report can be opened
via:

```sh
npm run bench:open
```
